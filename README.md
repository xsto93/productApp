# productApp
Prueba Técnica Zara.

Scrips de la aplicación

### `npm run start`
Para levantar el proyecto
```sh
npm run start
```

### `npm run test`
Comando para lanzar los test de src
```sh
npm run test
```
### `npm run build`
Comando para generar la build del proyecto
```sh
npm run build
```

### `npm run lint`
Comando para visualizar los errores de eslint
```sh
npm run lint
```
